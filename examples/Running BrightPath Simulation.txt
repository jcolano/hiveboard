 Running BrightPath Locally — Step by Step

  1. Start the backend server

  Open a terminal in C:\code\hiveBoard\src:

  cd C:\code\hiveBoard\src
  uvicorn backend.app:app --port 8000 --reload

  Wait until you see Uvicorn running on http://127.0.0.1:8000. Verify with:

  curl http://localhost:8000/health

  You should get {"status":"ok","version":"0.1.0"}.

  2. Run the simulator (simulation mode)

  Open a second terminal in C:\code\hiveBoard:

  cd C:\code\hiveBoard
  python examples/brightpath.py

  This runs all 6 agents at 1x speed with simulated LLM calls against localhost:8000 using the dev API key.

  3. Speed it up

  For faster cycles (great for quick demos):

  python examples/brightpath.py --fast

  Or custom speed:

  python examples/brightpath.py --speed 3

  4. Run a subset of agents

  python examples/brightpath.py --agents scout,harper,dispatch

  5. Enable real Haiku LLM calls

  This makes actual Anthropic API calls — real tokens, costs, latency, and generated content:

  python examples/brightpath.py --haiku

  The API key is auto-loaded from C:\code\apikeys\api_anthropic.key. You can also pass it explicitly:

  python examples/brightpath.py --haiku --anthropic-key sk-ant-...

  6. Open the dashboard

  Open your browser to:

  http://localhost:8000/static/index.html?apiKey=hb_live_dev000000000000000000000000000000

  You'll see all 6 agents with live telemetry: tasks, plans, LLM calls, issues, LoopColony posts, and heartbeats.

  7. Stop the simulator

  Press Ctrl+C in the simulator terminal. It gracefully shuts down all agents and flushes remaining events.

  ---
  Running against production

  From local machine:
  python examples/brightpath.py --endpoint https://mlbackend.net/loophive --api-key <your-prod-key>

  From production server (C:\hiveBoard):
  python examples/brightpath.py --endpoint http://localhost:8451 --api-key <your-prod-key>
